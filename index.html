💡 使用提示

🚀 启动开发服务器（推荐体验完整功能）：

npm run dev
# 然后访问 http://localhost:5173

🔧 编辑内容：在 Obsidian vault 中编辑 Markdown，保存即热更新。

📥 摄入新资料：放入 raw/ → 让 AI 执行 Ingest → 自动更新 wiki。

🏗️ 构建静态站点：

npm run build
# 构建产物在 Obsidian 仓库 .vitepress/dist/
